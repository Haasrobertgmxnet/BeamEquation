Detailed Description

A Physics-Informed Neural Network (PINN) model for 1D Euler-Bernoulli beam deflection.

Network architecture: Input: x ∈ ℝ¹ Hidden: two fully-connected layers with tanh activation Output: u(x) ∈ ℝ¹
