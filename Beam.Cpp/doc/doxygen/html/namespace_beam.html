Computes the boundary condition loss for the beam.

Enforces:

  • u'(0) = 0

◆ physics_loss()

torch::Tensor Beam::physics_loss ( PINN & model,
torch::Tensor input,
float EI = 1.0f )

Computes the physics loss (residual of the Euler-Bernoulli beam equation).

The differential equation: EI * d⁴u/dx⁴ = q(x) where q(x) = 1 is used here.

Parameters
modelThe PINN model
inputSample input points (x ∈ ℝ^{N×1})
EIBending stiffness coefficient (default = 1.0)
Returns
MSE of the PDE residuals
