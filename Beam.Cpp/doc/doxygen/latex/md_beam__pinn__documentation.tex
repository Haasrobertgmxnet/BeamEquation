 This repository implements a Physics-\/\+Informed Neural Network (PINN) to solve the Euler-\/\+Bernoulli beam equation. It combines machine learning (Lib\+Torch/\+C++) with physical constraints to estimate the deflection of a beam on the domain (\mbox{[}0, 1\mbox{]}).\hypertarget{md_beam__pinn__documentation_autotoc_md1}{}\doxysection{\texorpdfstring{Contents}{Contents}}\label{md_beam__pinn__documentation_autotoc_md1}
